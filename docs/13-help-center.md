## A) Home (`support-help-center-home.png`)
- Left **category sidebar:** getting started, social media connections, social media scheduling, media limits and processing, account and billing, troubleshooting, api, external integrations, faq.
- Top **search bar:** "Start typing to search an article, collection…" (`Ctrl K`).
- **Category grid cards** (icon + title + article count + blurb).
- **Featured articles** list (e.g. "Request a refund (how to)", "Finding TikTok Drafts Saved Outside the App").

## B) Article page (`support-connecting-linkedin.png`, `support-request-refund.png`)
- Left category nav (same), center article body (title + headings + bullet steps + inline links), right **"On this page"** TOC.
- Footer: **"Was this helpful?"** emoji reactions.
- Example content captured:
  - *Connecting LinkedIn accounts and pages* — account connection requirements, subpage functionality, "what this means for you".
  - *Request a refund (how to)* — "7-day 100% refund, no questions asked"; steps: log in → dashboard `/dashboard/refund` → submit → plans/addons cancel immediately → funds back in 5–12 business days; support email `support@post-bridge.com`.

## Suggested data model / API
```
HelpCollection: { id, slug, title, icon, articleCount, blurb }
HelpArticle:    { id, slug, collectionId, title, bodyHtml, headings:[], featured:bool }
```
- `GET /help/collections`, `GET /help/articles?collection=`, `GET /help/articles/:slug`, `GET /help/search?q=`.

## Notes for the clone
- Lowest priority for a functional clone — can be a static docs site or a 3rd-party help tool.
- The in-app **Docs** and **Get help connecting your accounts** links target this site.
